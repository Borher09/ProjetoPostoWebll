# ProjetoPostoWebll

Sistema de gerenciamento de postos de combustível que permite cadastrar, consultar e comparar os postos da cidade de Ji-Paraná. O aplicativo possibilita que o usuário visualize informações como nome do posto, endereço e valores dos combustíveis, além de oferecer um recurso de filtragem para identificar rapidamente o posto com o menor preço disponível.
